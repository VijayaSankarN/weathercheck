
Module: Weather Check
Author: Vijaya Sankar N <https://www.drupal.org/user/3238637>


Description
===========
Adds the Weather Check block to your website.


Installation
============
Copy the 'weathercheck' module directory in to your Drupal
sites/all/modules directory as usual.


Usage
=====
In the settings page enter latitude and longitude of the desired location or select the location from the map. Check the required parameters.
